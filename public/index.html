Förbrukningsrapport garaget Uppfinnaren -> MBF CSV

Ladda upp en Consumption Summary (User) .xlsx. Filen läses i webbläsaren, base64-kodas och skickas till /api/parse?format=json (lokalt: npm run dev).

MBF / IMD (avläsning)

Vid Convert to MBF CSV hämtas kolumnerna Start och Slut automatiskt från rapportfälten Date from och Date to i Excel-filen (som YYYYMMDD). RFID hämtas ur användarnamnet, t.ex. Admin RFID (AB1E9E41). Efter export visas Ladda ner MBF CSV under resultatet.

Mail MBF CSV skickar MBF-filen som bilaga via Resend. På servern krävs RESEND_API_KEY och RESEND_FROM_EMAIL (t.ex. i Vercel Environment Variables).
