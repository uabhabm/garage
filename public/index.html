Förbrukningsrapport → CSV

Ladda upp en Consumption Summary (User) .xlsx. Filen läses i webbläsaren, base64-kodas och skickas till /api/parse?format=json (lokalt: npm run dev).

MBF / IMD (avläsning)

Vid Convert to MBF CSV används start- och slutdatum som kolumnerna Start och Slut (format YYYYMMDD i filen). RFID hämtas ur användarnamnet, t.ex. Admin RFID (AB1E9E41).
